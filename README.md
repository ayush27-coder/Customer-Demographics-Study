# Customer Demographics Study

## Project Overview
This project focuses on analyzing customer demographic data to understand customer characteristics, preferences, and distribution patterns. The dashboard provides valuable insights into customer segments based on age, gender, income, location, and other demographic attributes.

## Objective
The main objective of this project is to:
- Analyze customer demographic characteristics.
- Identify key customer segments.
- Study age and gender distribution.
- Understand income and regional patterns.
- Support data-driven marketing and business decisions.

## Tools Used
- Tableau
- Microsoft Excel
- GitHub

## Dataset Information
The dataset contains customer demographic information including:

- Customer ID
- Age
- Gender
- Income
- Occupation
- Education Level
- Marital Status
- Region

## Key Performance Indicators (KPIs)
- Total Customers
- Average Age
- Average Income
- Gender Ratio

## Dashboard Features

### 1. Age Distribution Analysis
Visualizes customer distribution across different age groups.

### 2. Gender Distribution
Shows the proportion of male and female customers.

### 3. Income Analysis
Displays income patterns and customer purchasing potential.

### 4. Regional Distribution
Analyzes customer concentration across different regions.

### 5. Education and Occupation Analysis
Provides insights into customer educational backgrounds and professions.

## Key Insights
- The majority of customers belong to the working-age population.
- Customer distribution varies across regions.
- Income levels influence customer purchasing behavior.
- Gender demographics help identify target audiences.
- Education and occupation data support market segmentation.

## Conclusion
The Customer Demographics Study Dashboard helps businesses understand their customer base more effectively. By analyzing demographic characteristics, organizations can improve marketing strategies, customer targeting, and business decision-making.
